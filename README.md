# emoji-rain
emoji rain on your terminal

# Description
emoji-rain can rain emojis on your terminal.
Default emoji is 🐤.

```bash
$ npm emoji-rain

                                                                  🐤
                            🐤
                                                                                              🐤
                                                                🐤
                                                                                    🐤
                      🐤

                                                                                          🐤

                    🐤
                                                                                          🐤
                                                                  🐤
                                        🐤
                                                                                      🐤
                                                    🐤


          🐤
                                                                      🐤
🐤
                                                                                            🐤

                                                  🐤
                                          🐤
  🐤
                                                                        🐤
                                    🐤

```

### -e
`-e` can specify a character

```bash
$ npm emoji-rain -e 🍺

                                                                                                              🍺
                                                                                  🍺
                                                    🍺
                                                                                                      🍺
      🍺
                                                          🍺
                         🍺
                                                                                                   🍺
       🍺
                                                                                                                 🍺
                                                                                🍺
                                                                                                                        🍺
                                                                                                                      🍺
                                                                                                                  🍺
                                                                                            🍺
                                                                                                                              🍺
                                                                     🍺
   🍺
                      🍺
                                                                                        🍺
                                                                      🍺


```

### -s
`-s` can change the falling speed of letters.
Lower values ​​are faster.

# installation
```bash
$ npm install emoji-rain
```
