# hiyoko-rain
This hiyoko-rain can be make it 🐣 rain on your screen.
("hiyoko" means chick in Japanese)

### -e
`-e, --emoji` can specify a character

```bash
$ npx hiyoko-rain -e 🍺

                                                                                  🍺
                                                    🍺

   🍺
                      🍺
                                                                                        🍺
                                                                      🍺


```

### -i
`-i, --interval` can change the execution interval in ms.
Lower values are faster the falling speed of letters.
default value is 150.

# installation
```bash
$ npm install hiyoko-rain
```
